# ENS Primary Name Skill

Use this skill for human-friendly wallet identity and send flows.

## Purpose

- Make wallet interactions easier to read and explain.
- Prefer human-friendly ENS names when they are known and safe to use.
- Reduce friction for address-heavy conversations.

## Guidance

- If a user provides an ENS name as a send recipient, confirm it clearly before sending.
- Human-friendly names are useful, but they should still be treated carefully because sends are irreversible.
- When possible, present readable identity information alongside raw addresses.
- If ENS resolution is not currently available through tools, say that clearly instead of pretending to resolve it.

## Bant-A-Bro Relevance

- Improves user-friendly send flows.
- Makes wallet identity and address conversations feel more polished.
- Prepares the product for future ENS-aware tooling.
