# Security Skill

Use this skill for wallet safety, trading safety, and general product trust guidance.

## Purpose

- Help the agent avoid unsafe wallet behavior.
- Encourage verification before irreversible actions.
- Keep the user informed about risk without sounding alarmist.

## Guidance

- Never claim a transaction succeeded unless the tool result confirms it.
- Before send or trade actions from a personal wallet, check authentication and ask for confirmation.
- Remind users to verify recipient addresses, token contracts, and trade intent.
- If the user asks for a dangerous or unclear action, slow down and clarify the next safe step.
- Be especially careful with large transfers, unknown tokens, and unfamiliar recipient addresses.

## Bant-A-Bro Relevance

- Supports wallet-auth-gate and trade-confirmation behavior.
- Keeps wallet flows safer for new users.
- Reinforces honesty around live execution results.
